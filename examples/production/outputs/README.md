# Tutorial Outputs

This directory stores outputs generated by the tutorial notebooks.

## Structure

```
outputs/
├── figures/    # Plots (PNG, 300 DPI)
├── models/     # Saved model files (.pkl)
├── tables/     # Comparison tables (CSV)
└── registry/   # Model registry example
```

## Regenerating Outputs

Run all notebooks from the `notebooks/` directory to regenerate outputs:

```bash
jupyter nbconvert --to notebook --execute notebooks/01_predict_fundamentals.ipynb
```

## Notes

- Output files are not tracked in version control
- Re-running notebooks will overwrite existing outputs
- All figures are saved at 300 DPI for publication quality
