# Outputs Directory

## Purpose

This directory stores outputs generated by running the tutorial notebooks.

## Structure

```
outputs/
├── figures/
│   ├── irfs/              # Impulse response function plots
│   ├── fevds/             # Forecast error variance decomposition plots
│   ├── causality_networks/ # Granger causality network diagrams
│   ├── diagnostics/       # Residual diagnostic plots
│   ├── forecasts/         # Forecast fan charts
│   ├── vecm/              # VECM-related figures
│   └── gmm/               # GMM estimation figures
├── tables/                # Summary tables (CSV format)
└── results/               # Numerical results (JSON format)
```

## Important Notes

- These files are **generated automatically** when running notebooks. Do not edit manually.
- To regenerate outputs, re-run the corresponding notebook.
- Figures are saved as PNG at 150 DPI by default.
- Tables are saved as CSV for easy loading in subsequent analyses.
