# Diagnostics Tutorial Outputs

**Version:** 1.0.0
**Last Updated:** 2026-02-22

This directory stores all outputs generated by running the diagnostics tutorial notebooks, including figures, tables, and saved test results.

---

## Purpose

Outputs are organized by type and tutorial to facilitate:
- Easy retrieval and reference
- Reproducibility verification
- Comparison across tutorials
- Reuse in presentations or reports

**Note:** This directory is typically excluded from version control (`.gitignore`) since outputs can be regenerated from notebooks. Directory structure is preserved with `.gitkeep` files.

---

## Naming Conventions

### Figures

**Format:** `{notebook}_{figure_type}_{description}.png`

**Examples:**
- `01_unit_root_time_series_gdp.png`
- `01_unit_root_test_comparison_barplot.png`
- `02_cointegration_residuals_oecd.png`
- `03_specification_hausman_scatter.png`
- `04_spatial_moran_scatterplot.png`
- `04_spatial_lisa_cluster_map.png`

### Tables

**Format:** `{notebook}_table_{number}_{description}.{ext}`

**Extensions:**
- `.csv`: For programmatic use and further analysis
- `.tex`: For LaTeX documents
- `.html`: For web viewing

**Examples:**
- `01_unit_root_table_01_llc_results.csv`
- `02_cointegration_table_02_pedroni_stats.csv`
- `03_specification_table_01_hausman_test.csv`
- `04_spatial_table_01_moran_i_results.csv`

### Results

**Format:** `{notebook}_{test_name}.json`

**Examples:**
- `01_unit_root_ips_test.json`
- `02_cointegration_westerlund_ecm.json`
- `03_specification_hausman_test.json`
- `04_spatial_lm_tests.json`

## Figure Specifications

- **Format:** PNG (recommended for quality and compatibility)
- **Resolution:** 300 DPI (publication quality)
- **Size:** Typically 8x6 inches (standard) or 12x8 inches (wide plots)

**Creating publication-quality figures in notebooks:**
```python
import matplotlib.pyplot as plt

fig, ax = plt.subplots(figsize=(8, 6), dpi=300)
# ... plotting code ...
plt.tight_layout()
plt.savefig(
    '../outputs/figures/01_unit_root/time_series_gdp.png',
    dpi=300,
    bbox_inches='tight',
    facecolor='white'
)
plt.close()
```

## Regenerating Outputs

All outputs can be regenerated by running the notebooks:

```bash
# Single notebook
jupyter nbconvert --to notebook --execute notebooks/01_unit_root_tests.ipynb

# All notebooks
for notebook in notebooks/*.ipynb; do
    jupyter nbconvert --to notebook --execute "$notebook"
done
```

## Output Management

### Cleaning Outputs

```bash
# Remove all generated outputs
rm -rf outputs/figures/*/*.png
rm -rf outputs/tables/*/*.csv
rm -rf outputs/tables/*/*.tex
rm -rf outputs/results/test_results/*.json
rm -rf outputs/results/diagnostics/*.json

# Keep directory structure
find outputs -type d -empty -exec touch {}/.gitkeep \;
```
